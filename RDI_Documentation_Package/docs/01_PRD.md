# Product Requirements Document

## MVP
- Discover Redis OSS, Enterprise, Valkey, KeyDB where possible
- Auto-detect local subnets
- User-defined CIDRs and ports
- TLS support
- Anonymous/authenticated PING
- Read-only inventory
- CLI + Web UI
- CSV export

## Non-Goals
No writes, no credential storage, no brute-force, no scheduled scans, no command console (post-MVP).
